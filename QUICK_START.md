# 🚀 Quick Start - India News App

## ⚡ **5-Minute Setup**

### **Step 1: Get Your FREE API Key (2 minutes)**

1. Go to: **https://newsdata.io/register**
2. Sign up with your email
3. Verify email
4. Copy your API key from dashboard

---

### **Step 2: Add API Key (1 minute)**

**Open:** `app/src/main/java/com/example/indianews/utils/Constants.java`

**Line 12 - Replace:**
```java
public static final String NEWS_API_KEY = "YOUR_NEWSDATA_IO_API_KEY_HERE";
```

**With your key:**
```java
public static final String NEWS_API_KEY = "pub_YOUR_ACTUAL_KEY";
```

**Save** (Ctrl+S)

---

### **Step 3: Build & Run (2 minutes)**

1. Open project in Android Studio
2. Wait for Gradle sync
3. Click **Run** ▶️
4. Select device
5. **Done!** 🎉

---

## 📱 **Features You'll See:**

✅ **🔥 Trending Carousel** - Auto-scrolling top stories
✅ **📍 City Selector** - Choose from 10 Indian cities
✅ **📰 Categories** - Politics, Tech, Sports, etc.
✅ **🌍 World News** - Tap chip to switch
✅ **⭐ Favorites** - Save articles for later
✅ **📖 Full Articles** - Read in-app (WebView)

## 🎯 **Usage:**

- **Trending** → Tap any card in carousel
- **City** → Select from dropdown (Mumbai, Delhi, etc.)
- **Categories** → Tap chips to filter
- **Article** → Tap to read → "Read Full Article" for complete content
- **Favorite** → Heart button to save
- **Search** → Top-right search icon

## ❓ **Troubleshooting:**

**"No news available"**
- Check API key is correct
- Check internet connection
- Free plan has 12-hour delay (yesterday's news)

**Build errors**
- **Build → Clean Project**
- **Build → Rebuild Project**

**App crashes**
- Make sure API key is not "YOUR_NEWSDATA_IO_API_KEY_HERE"
